1. Introduction

CellDiff is a Microsoft Office Excel add-in to compare cell contents. See the screenshot below.

Fig. 1: Screenshot of compared cells using CellDiff

1.1 Background

I wrote this program to facilitate document review process.

When I review some documents, I am often asked to write the review comments and suggested changes on an Excel sheet. Such a form usually contains columns such as: line number, original sentence, suggested change, and comments. The original author or a document editor reads the comments and decide whether to apply the suggested changes. I believe it is a common practice.

If a change to a particular sentence is small, e.g., just a correction of a typo or two, the original sentence and suggested change are almost same, and I often need to explain which letter I changed in the comment column. I considered it is a waste of time, and created this small add-in to avoid it.

2. How to use

On the Excel's ribbon, under the Review tab, you see a large command icon labeled Compare Cell Contents. (See the screenshot below.) Select cells that contain texts you want to compare, then click on the Compare Cell Contents. Contents of the sellected cells are compared, and the differences are emphasized.

Fig. 2: Ribbon command icon of CellDiff

2.1 Using Compare Cell Contents

There are five ways to select cells for Compare Cell Contents:

  • Select two adjacent cells with a single mouse dragging. This is the most basic way to use Compare Cell Contents. Two cells may be adjacent horizontally (i.e., they are on a same row) or vertically (i.e., they are on a same column.)
  • Select several rows in two adjacent columns with a single mouse dragging, resulting a rectangular selection of width two. Two cells on a same row will be compared in this case.
  • Select several columns in two adjacent rows with a single mouse dragging. This is a swapped row-column version of the above. Two cells on a same column will be compared in this case.
  • Select two non-adjacent cells, by clicking on the first cell and then control-clicking on the second cell. Then, click on Compare Cell Contents.
  • Select two separate (non-adjacent) areas of cells. This is usuful if you want to compare non-adjacent cells. Select the first group of cells, as usual, with a single mouse dragging, then control-drag over the second group of cells. This way, you can get two separate areas of cells selected. Click on Compare Cell Contents, then. Each cell in the first area are compared against a cell in the second area. Note that the two areas should contain the same number of cells.

2.2 Using Compare with Advanced Options

To the lower right of Compare Cell Contents command icon is a dialog box launcher symbol.

Fig. 3: Dialog box launcher synbox (pointed to by a red arrow)

As other dialog box lancher symbols appearing elsewhere on the ribbon, when you click on it, a dialog box opens so that you can specify some detailed and/or advanced options when comparing cells. With this dialog box, you can instruct CellDiff to behave differently than what you can get with Compare Cell Contents.
